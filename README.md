# 📊 Excel Data Cleaning & Dashboard Project

## 📌 Objective
To clean raw sales data and create a dashboard for better insights.

## 🧹 Data Cleaning Steps
- Removed duplicate records
- Handled missing values
- Standardized data formats
- Created calculated columns (Profit, Delivery Days)

## 📊 Dashboard
- Sales by Region (Bar Chart)
- Sales by Category (Pie Chart)
- Total Sales & Profit summary

## 🛠️ Tools Used
- Microsoft Excel
- Pivot Tables
- Data Cleaning Functions
